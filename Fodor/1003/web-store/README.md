# Web Store Project

## Overview
This web store project is a simple e-commerce application that allows users to browse items, add them to a shopping cart, and view their selections. The project is structured with separate files for HTML, CSS, JavaScript, and data.

## Project Structure
```
web-store
├── public
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── script.js
│   └── index.html
├── src
│   ├── data
│   │   └── items.json
│   └── app.js
└── README.md
```

## Features
- Display a list of items available for purchase.
- Add items to a shopping cart.
- View the contents of the shopping cart.
- Responsive design for mobile and desktop views.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.)
- A local server (optional, for testing)

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd web-store
   ```

### Running the Project
1. Open `public/index.html` in your web browser to view the web store.
2. If using a local server, start the server and navigate to the appropriate URL.

## Usage
- Browse through the available items.
- Click on "Add to Cart" to add items to your shopping cart.
- View your cart to see selected items.
