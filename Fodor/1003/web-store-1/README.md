# Web Store

This project is a simple web store that allows users to view available items and manage a shopping cart. It is built using HTML, CSS, and JavaScript.

## Project Structure

```
web-store
├── css
│   └── styles.css       # Styles for the web store
├── js
│   └── script.js        # JavaScript functionality for the web store
├── index.html           # Main HTML document for the web store
└── README.md            # Project documentation
```

## Features

- View a list of available items
- Add items to the shopping cart
- View the total price of items in the cart

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the web store.

## Usage Guidelines

- Click on the items to add them to your cart.
- Navigate to the cart section to view your selected items and the total price.
